# BrokeBites

> ⚠️ **Note**: This project is currently a work in progress.

## About

BrokeBites is a social meal sharing platform where friends can discover and share their daily meals. It's designed to provide meal inspiration while making it easy to see what ingredients go into each dish.

## Current Features

- Share photos of your meals with ingredients lists
- Like and comment on friends' posts
- Social feed of meal inspirations
- View detailed ingredient information

## Tech Stack

Built with:
- Flutter
- Dart

## Setup

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
